open terminal here to run the commands
-to run the app
node app.js

-to add a note
node app.js add --title="note name" --body="content of the note"

-to remove a note
node app.js remove --title="note name"

-to list all the notes
node app.js list

-to read a particular note
node app.js add --title="note name"

if npm modules are missing
npm install
